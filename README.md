# 经典扫雷在线游戏

Windows 经典扫雷游戏的网页复刻版，可在电脑或手机浏览器中直接游玩，无需下载安装。

## 功能特性

- **5 种难度级别**：基础 (9x9)、中级 (16x16)、专家 (自适应屏幕)、满屏（全屏模式）、自定义
- **跨平台兼容**：支持鼠标操作和触屏操作（长按插旗、短按开格）
- **本地持久化**：自定义棋盘设置保存在 `localStorage` 中
- **经典玩法还原**：左键揭开、右键插旗/解除、数字 chord 连点揭开

## 文件说明

| 文件 | 说明 |
|---|---|
| `index.html` | 游戏主页面，包含难度选择面板和游戏画布 |
| `game.js` | 核心游戏逻辑：棋盘生成、雷区展开、胜负判定、计时器 |
| `main.js` | 资源加载与 UI 事件绑定 |
| `assets.js` | 游戏图片资源（图标、数字贴图等） |
| `styles.css` | 全局样式与移动端适配 |

## 操作说明

| 操作 | 效果 |
|---|---|
| 左键点击格子 | 揭开格子 |
| 右键点击格子 | 插旗 / 解除插旗 |
| 已揭开的数字格左键 | Chord：若周围旗帜数等于数字，自动揭开周围未插旗的格子 |
| 长按屏幕 | 触屏模式下触发插旗 |
| 短按屏幕 | 触屏模式下揭开格子 |
| 点击中间笑脸 | 重新开始 |

## 难度参数

| 级别 | 宽 | 高 | 雷数 |
|---|---|---|---|
| 基础 | 9 | 9 | 10 |
| 中级 | 16 | 16 | 40 |
| 专家 | 16/30 | 16/30 | 99 |
| 满屏 | 自适应窗口 | 自适应窗口 | 按密度公式计算 |
| 自定义 | 用户输入 | 用户输入 | 用户输入 |

## 运行方式

直接用浏览器打开 `index.html` 即可开始游戏。

```
file:///路径/to/index.html
```

也可通过任意静态 HTTP 服务器托管：

```bash
# Python
python3 -m http.server 8000

# Node.js (npx)
npx serve .
```

然后访问 `http://localhost:8000`。
